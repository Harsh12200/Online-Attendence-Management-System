# User Management Web Application

## Overview
This project includes HTML templates for user management including login, registration, and user profile pages. It is styled with CSS and Bootstrap and features JavaScript-based form validation for a professional user experience.

## Features
- **Responsive Design**: Templates are fully responsive and work across devices.
- **Form Validation**: Email and password validation with real-time feedback.
- **Modern UI**: Clean, modern design using Bootstrap.

## Setup Instructions
1. Clone this repository to your local machine.
2. Open the HTML files in any browser to view the templates.
3. For local development, you can serve the pages via any HTTP server (e.g., VS Code Live Server extension).
